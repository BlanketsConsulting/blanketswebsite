# Blankets Consulting Group — Website

Professional website for Blankets Consulting Group, a boutique business consulting firm in Rochester, MN offering fractional CxO services, strategic marketing, and digital growth solutions.

## Pages

| Page | Path | Description |
|------|------|-------------|
| Homepage | `/` | Hero, services, results, industries, CTA |
| Services | `/services` | Three-tier service packages |
| About | `/about` | Jorge Mosquera's story & credentials |
| Results | `/results` | Client case studies & testimonials |
| Industries | `/industries` | Six industry verticals |
| Insights | `/insights` | Blog/articles grid |
| Contact | `/contact` | Contact form & info |
| Diagnostic | `/diagnostic` | Booking landing page |

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks, no build step)
- Google Fonts: DM Serif Display + Plus Jakarta Sans
- Responsive: Desktop, Tablet, Mobile
- SEO: JSON-LD schema, meta tags, semantic HTML

## Deployment

Static site — upload directly to any web host. No build process required.
